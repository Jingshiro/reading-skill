# Reading Skill

一个 Claude Agent Skill，整合多个阅读平台的数据，提供书架管理、笔记整理、阅读统计与书籍推荐。

## 支持平台

| 平台 | 接入方式 |
|------|----------|
| [Legado](https://github.com/Jingshiro/legado) | 本地 Web 服务 |
| [Reeden](https://reeden.app/) | 本地数据解析（备份文件） |
| 微信读书 | 远程 API（需 `WEREAD_API_KEY`，参见 [weread-skills](https://cdn.weread.qq.com/skills/weread-skills.zip)） |

## 能做什么

- 📚 查看书架、搜索书籍
- 📝 汇总划线、批注和想法
- 📊 分析阅读时长、偏好标签、阅读习惯
- 🎯 生成阅读画像
- 💡 根据偏好推荐相似书籍
- 🔗 多平台数据合并去重
